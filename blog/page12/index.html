机械工程师的级别

首先，千万不要以为机械只是搞结构，更不要以为结构只是使用 PROE、solidworks 或者其他什么软件。其实搞机械机构的有个先天优势，那就是什么产品都是以结构为平台的，你往哪里发展都有可能。搞结构能让你对产品有更直观的理解，让你更严谨，更切实。另一方面，搞结构的是个漫长痛苦的锻炼过程，但是一旦练就本领，多老都不怕，甚至越老越吃香。

根据我自己的从业经历，我随便列几条我认为搞结构的人的发展路径（个人看法，），依次为：

菜鸟1级：会使用 AUTOCAD 画一些简单的二维图，根据这个图，你能想象产品的三维形态。

菜鸟2级：你的二维图上能够看到公差，材料，表面处理，工艺技术要求等信息了。而且这些信息是你自己标上去的，你知道他们意味着什么。

活在当下

走到生命的哪一个阶段，都该喜欢那一段时光，完成那一阶段该完成的职责，顺生而行，不沉迷过去，不狂热地期待着未来，生命这样就好。不管正经历着怎样的挣扎与挑战，或许我们都只有一个选择：虽然痛苦，却依然要快乐，并相信未来。白岩松

人生要活在当下，及时享受每一天。

人生其实是很短暂的，更不用提由于生命的脆弱而发生的各种意外。而且生活本身就是过程，结果只是短暂的时刻。所以，当清晨睁开第一眼的瞬间，就应该翘起嘴巴露出发自内心的微笑，因为至少我们还活着，又可以享受明媚的阳光，呼吸新鲜的空气，听到欢快的鸟语，问到沁人心脾的花香……生活到处都是美，只是由于我们迫于追赶时代而暂时蒙蔽了发现美的眼睛。活在当下，就不要再为未来做任何毫无意义的纠结、恐惧了。该来的迟早要来，躲也躲不过去，既不会提前也不会推迟；不该来的再强求也不会来。我们所能做的只有把握好当下，把当下该做好的事情做好，仅此而已。所以，为永远无法预测的事情枉费心机，浪费感情真是一件得不偿失的事情，将这部分时间精力节省出来更好的享受当下不更好！

Photoshop CS5 学习笔记

基础知识

1、PS CS5 界面组成部分：视图控制栏、菜单栏、工具箱、工具选项栏、控制面板、工作区域。

2、主窗口中一般只会显示一些最常用的控制面板，比如颜色、、图层、管道等，可以在菜单栏的“窗口”菜单中勾选相关的控制面板使其在主窗口中显示出来。

3、放大与缩小：a、工具箱放大镜工具（Z）；b、Ctrl+“+”、Ctrl+“-”；c、Alt+鼠标滚轮；d、Ctrl+0（满画面显示图片）。

4、手抓工具：a、工具箱中的手抓工具（H）；b、按住空格键的同时拖动鼠标左键。

5、自由变换：Ctrl+T。可以实现图片的缩放、旋转等操作。在按住 Shift 的时候拖动图片的四个角点可以实现等比缩放。自由变换之后确认可以按“Enter”，取消自由变换则“Esc”。

Inkscape 学习笔记

操作相关

1. 改变画布大小使其自动调整以适应内容大小

File → Document Properties → Page → +Resize page to content… → Resize page to drawing or selection

2. 矩形

矩形绘制技巧：

  • 配合 Ctrl 键，可以绘制正方形或边长为整数比率(2：1，3：1等)的矩形。
  • 配合 Shift 键，从中心开始绘制。

刚绘制的矩形(新绘的矩形总是自动被选中)在三个角上显示3个控制柄，但实际上有四个控制柄，如果矩形没有圆角，右上角的两个是重合在一起的。这两个是倒圆控制柄；另外两个(左上和右下)是缩放控制柄。

任务描述

获取一个目录里面的所有文件的文件名（带扩展名），不带扩展名的文件名，以及扩展名。

脚本程序

1     #! /bin/bash 
2     
3     for file in `ls $1` 
4     do 
5     filename=${file%.*}
6     extension=${file##*.}
7     echo $filename
8     echo $extension
9     done

CSS伪类选择器

基本介绍

CSS3 的伪类选择器有多，本文仅讨论状态伪类选择器——:hover、:active、:focus。这些选择器都有一个共同的特征名那就是定义的样式只有当元素处于某种状态下时才起作用，在默认状态下无效。

  • :hover选择器：当鼠标悬停在所指定的元素上时所使用的样式；
  • :active选择器：当所指定的元素处于激活状态（鼠标在元素上按下还没有松开）时所使用的样式；
  • :focus选择器：当元素获得光标焦点时使用的样式，主要用在文本框输入文字时使用；

状态选择器最常用的地方就是链接和输入框了。其中的 :focus 伪选择器主要用在键盘操作者进行页面的访问，利用 Tab 键即可查看到当某链接为选定的链接后的效果。
